# 💫 About Me:
I'm Harsh Mehta, a B.Tech Computer Science & Engineering student at IILM University (2023–present) with a strong passion for building full-stack, real-time, and AI-powered web applications. I specialize in modern tech stacks including React, Next.js, Node.js, Express, MongoDB, Tailwind CSS, and Shadcn UI, along with real-time systems (Socket.IO), authentication (JWT, Clerk), cloud services (Cloudinary, ImageKit), and AI tools like RAG pipelines, LangChain, Prompt Engineering, and NLP.<br>Through hands-on projects, I've created:<br><br>VIBEON – a complete full-stack social media platform with posts, likes, comments, real-time notifications, and secure JWT authentication.<br>CREON – an AI-powered content management system (CMS) for creators featuring rich text editing, AI-assisted workflows, and real-time data sync via Convex.<br>BeThere – an AI-driven SaaS event management platform focused on seamless creation, management, and user engagement.<br><br>I also completed an internship at YBI Foundation, where I developed a Twitter Sentiment Analysis project using NLP and gained practical exposure to Generative AI foundations. In addition, I hold certifications in Python and C from Infosys, Machine Learning Foundations from AWS Academy, and successfully completed a 22-week DSA Challenge on GeeksforGeeks, solving 160 problems.<br>I love turning ideas into clean, scalable, and user-friendly products.Ready to connect and collaborate!<br><br>
